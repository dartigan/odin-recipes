# Odin Recipes

This is a simple recipe website created as part of **The Odin Project** course:  
https://www.theodinproject.com/

The goal of this project is to practice **HTML fundamentals**.

---

## Project Description

This project contains:

- A main `index.html` file, serving as the entry point to the website
- An `images/` folder to store images used in the project
- A `recipes/` folder containing individual recipe pages written in HTML

---

## Skills Showcased

- Semantic HTML
- Use of relative and absolute paths for files and images
- HTML lists, paragraphs, and heading elements
- Embedding images in HTML pages
